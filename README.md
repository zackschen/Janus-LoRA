# Janus-LoRA

This repository will contain the official implementation of **Janus-LoRA**, a unified framework for LoRA-based exemplar-free continual learning.

Janus-LoRA is designed to balance two key goals in continual learning:

- **Stability**: preserving knowledge from previous tasks.
- **Plasticity**: maintaining the ability to learn new tasks.

The method combines parameter-level update rectification with feature-level separation to reduce forgetting while supporting effective new-task adaptation.
